# Sushi Peripherals

## Prerequisites
- Foundry
- Make

## Getting Started
Initialize
```sh
make init
```

Copy `.env.example` to `.env` and set the RPCs for deploying contracts, then source `.env`

```sh
cp .env.example .env
source .env
```

Build & Test

```sh
make build
make test
```
